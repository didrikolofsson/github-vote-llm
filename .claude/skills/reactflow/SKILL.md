# React Flow (@xyflow/react)

React Flow is a library for building interactive, node-based UIs such as flowcharts, diagrams, visual programming tools, and workflows.

## How to Look Up Docs

Fetch `https://reactflow.dev/llms.txt` to get a full, up-to-date index of all React Flow guides, API reference pages, and examples — including direct URLs to every component, hook, type, and utility.

Use `WebFetch` on any URL from that index to get the full content of a specific page.

**Always fetch before writing non-trivial React Flow code.** The API has many subtle props and gotchas — don't guess.
